# Subtitle Bridge for Stremio

إضافة ترجمات شخصية لستريمي، بلا ذكاء اصطناعي وبلا روابط فيديو. تجمع:

- OpenSubtitles V3 الرسمي (مفعّل افتراضيًا، بلا مفتاح).
- SubDL API (اختياري؛ يحتاج مفتاح API مجاني).

ترتّب النتائج وفق تشابه اسم إصدار الترجمة مع اسم ملف الفيديو (`WEB-DL`، `BluRay`، الدقة، الترميز)، وتعطي نتائج SubDL المطابقة للموسم والحلقة فقط.

## النشر المجاني على Render

1. أنشئ مستودعًا **خاصًا** جديدًا في GitHub باسم `subtitle-bridge`.
2. ارفع محتويات هذا المجلد إلى المستودع. لا ترفع ملف `.env` حقيقي ولا أي مفتاح.
3. أنشئ مفتاحًا مجانيًا من [SubDL](https://subdl.com/panel/api).
4. في [Render](https://render.com/) اختر **New +** ثم **Blueprint**، واربط مستودع GitHub واختر `render.yaml`.
5. في صفحة Environment في Render، أضف `SUBDL_API_KEY` وضع المفتاح فيه. لا تضعه في GitHub أو رابط ستريمي.
6. اضغط Deploy. بعد انتهاء النشر انسخ عنوان الخدمة، مثل:
   `https://subtitle-bridge.onrender.com`
7. افتح هذا الرابط في المتصفح مع `/manifest.json` بالنهاية:
   `https://subtitle-bridge.onrender.com/manifest.json`
   ثم اختر **Install** في Stremio، أو أضف الرابط يدويًا من صفحة الإضافات.

## الإعدادات

| متغير Render | القيمة المقترحة | وظيفته |
| --- | --- | --- |
| `SUBTITLE_LANGUAGES` | `AR,EN` | اللغات المراد إظهارها |
| `SUBDL_API_KEY` | مفتاحك | يفعّل مصدر SubDL |
| `HIDE_HI` | `true` | يخفي ترجمات الوصف السمعي |
| `ENABLE_OPEN_SUBTITLES` | `true` | المصدر الرسمي OpenSubtitles |
| `ENABLE_SUBDL` | `true` | مصدر SubDL |

## ملاحظات مهمة

- لا توجد ترجمة أو توليد نصوص بالذكاء الاصطناعي في الكود.
- **الخيار المجاني الأفضل لهذه الإضافة: Koyeb**؛ لديه Free Instance دائم واحد وينام بعد ساعة من عدم الاستخدام. Render مجاني أيضًا، لكنه ينام بعد 15 دقيقة ولديه حد 750 ساعة شهريًا. إذا اخترت Koyeb: اربط المستودع، اختر Node.js، واجعل أمر التشغيل `npm start`، ثم أضف متغيرات البيئة نفسها في الجدول أعلاه.
- لا يمكن ضمان تزامن 100% إذا لم يمرّر المشغل اسم الملف أو بصمته؛ الإضافة تضع النتيجة الأقرب في الأعلى وتبقي البدائل ظاهرة.
- لا ترسل مفاتيح API في المحادثة. ضعها بنفسك داخل إعدادات Render فقط.
